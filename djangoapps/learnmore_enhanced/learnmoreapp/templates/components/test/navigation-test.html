Navigation Component Test

This page tests the navigation component's functionality and responsiveness.

Responsive Test

Resize your browser window to test the responsive behavior:

  • Desktop: Full navigation with all items visible
  • Tablet: Hamburger menu appears, navigation items in dropdown
  • Mobile: Compact layout with mobile menu

Dark Mode Test

Click the theme toggle button to test dark mode:

  • Light mode: White background, dark text
  • Dark mode: Dark background, light text
  • Hover states should work in both modes

Interaction Test

Test the following interactions:

  • Click the hamburger menu to open/close mobile navigation
  • Hover over navigation links to see hover states
  • Click the Sign In button to verify link behavior
  • Test keyboard navigation (Tab key)

Accessibility Test

Verify the following accessibility features:

  • Proper ARIA labels on all interactive elements
  • Keyboard navigation works correctly
  • Focus states are visible
  • Screen reader friendly structure
