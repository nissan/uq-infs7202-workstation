Introduction to Neural Networks

Neural networks are a set of algorithms, modeled loosely after the human brain, that are designed to recognize patterns. They interpret sensory data through a kind of machine perception, labeling or clustering raw input. The patterns they recognize are numerical, contained in vectors, into which all real-world data, be it images, sound, text or time series, must be translated.

Neural networks help us cluster and classify data. You can think of them as a clustering and classification layer on top of the data you store and manage. They help to group unlabeled data according to similarities among the example inputs, and they classify data when they have a labeled dataset to train on.

The Biological Inspiration

Neural networks, as their name implies, are inspired by the human brain's neural structure. The basic computational unit of the brain is a neuron. Approximately 86 billion neurons can be found in the human nervous system, and they are connected with approximately 10^14 - 10^15 synapses.

Figure 1: Comparison between biological neurons (left) and artificial neural networks (right)

Each neuron receives input signals from its dendrites and produces output signals along its axon. In our artificial neural networks, the signals that travel along the axons interact multiplicatively with the dendrites of the other neurons based on synaptic strength, or weights.

The Perceptron: The Building Block

The perceptron is the simplest neural network, comprising a single neuron. It was developed in the late 1950s by Frank Rosenblatt. A perceptron takes several inputs, multiplies each by a weight, sums them up, and passes the sum through an activation function to produce an output.

Mathematical Representation

The output of a perceptron can be represented mathematically as:

Python
def perceptron(inputs, weights, bias):
    # Calculate the weighted sum of inputs
    weighted_sum = sum(w * x for w, x in zip(weights, inputs)) + bias
    
    # Apply activation function (step function in this case)
    if weighted_sum > 0:
        return 1
    else:
        return 0

# Example usage
inputs = [0.5, 0.3, 0.2]
weights = [0.4, 0.7, 0.2]
bias = -0.5

output = perceptron(inputs, weights, bias)
print(f"Perceptron output: {output}")

The perceptron can only solve linearly separable problems, such as implementing logical AND and OR operations. However, it cannot solve the XOR problem, which is not linearly separable. This limitation led to the development of multi-layer perceptrons (MLPs).

Activation Functions

Activation functions introduce non-linearity into neural networks, allowing them to learn complex patterns. Without activation functions, neural networks would be equivalent to linear regression models, regardless of their depth.

Common activation functions include:

  • Sigmoid: Maps input to a value between 0 and 1. Useful for output layers in binary classification.
  • Tanh: Maps input to a value between -1 and 1. Similar to sigmoid but zero-centered.
  • ReLU (Rectified Linear Unit): Returns the input if positive, otherwise returns zero. Most commonly used in hidden layers.
  • Leaky ReLU: A variant of ReLU that allows a small gradient when the unit is not active.
  • Softmax: Used in the output layer for multi-class classification problems.

Training Neural Networks

Training a neural network involves adjusting the weights and biases to minimize the difference between the predicted outputs and the actual outputs. This is typically done using an algorithm called backpropagation, which computes the gradient of the loss function with respect to each weight.

The training process generally involves the following steps:

  1. Forward Propagation: Input data is passed through the network to generate predictions.
  2. Loss Calculation: The difference between predictions and actual values is measured using a loss function.
  3. Backward Propagation: The gradient of the loss with respect to each weight is calculated.
  4. Weight Update: Weights are adjusted in the direction that reduces the loss, typically using gradient descent or its variants.
Key Points to Remember
  • Neural networks are inspired by the structure and function of the human brain.
  • The perceptron is the simplest form of a neural network, consisting of a single neuron.
  • Activation functions introduce non-linearity, allowing networks to learn complex patterns.
  • Multi-layer networks can solve problems that single-layer networks cannot.
  • Backpropagation is the key algorithm for training neural networks.
  • Neural networks excel at pattern recognition tasks like image and speech recognition.
