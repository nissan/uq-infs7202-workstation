Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display the content.

HTML elements are represented by tags, written using angle brackets. Tags come in pairs, with the opening tag and the closing tag. The content goes between these tags.

Here are some key points about HTML:

  • HTML stands for HyperText Markup Language
  • HTML is used to create the structure of web pages
  • HTML consists of elements represented by tags
  • Tags usually come in pairs: opening tag and closing tag
  • The browser does not display the HTML tags, but uses them to render the content
Figure 1: Basic structure of an HTML document showing the relationship between different elements.
HTML
<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
    
    <h2>HTML Elements</h2>
    <ul>
        <li>Headings: h1, h2, h3, ...</li>
        <li>Paragraphs: p</li>
        <li>Lists: ul, ol, li</li>
        <li>Links: a</li>
        <li>Images: img</li>
    </ul>
</body>
</html>
